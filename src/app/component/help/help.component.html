Website Help Manual

  1. There are three parts of the website:
    • Customer Side.
    • Admin Side.
    • Restaurant Side.

  2. How to use the Food Delivery Website as a Customer:
    • Customer can view the items in the restaurant without registration and login, but will not be able to order it.
    • Customer has to register himself/herself in order to use the website.
    • After successful registration only the customer can log into the website.
    • After successful login, customer can view all the restaurants approved by the admin.
    • Customer can view all the cuisines and add them to cart if that item is available.
    • Customer cannot order if that item is not available.
    • After adding the items to the cart, customer can pay using credit card and order will be placed.

  3. How to use the Food Delivery Website as an Admin:
    • Admin will have certain privileges only reserved for the admin.
    • Admin has to login using the home page login form.
    • After successful login, admin will arrive to the main admin page.
    • Admin can view, update and delete any customer record if required.
    • Admin can view, update and delete any restaurant record if required.
    • Admin will approve restaurant whenever any new restaurant is added.

  4. How to use the Food Delivery Website as a Restaurant Owner:
    • Any new restaurant will have to regsiter in to the website.
    • After successful registration, restaurant owner will have to login using appropriate credentials.
    • After logging into his restaurant, items of the restaurant can be added.
    • Only after the admin approval, the restaurant will be displayed to the customer on the website
    • When adding items, every detail must be filled accordingly.
    • If item is not available, add to cart will be disabled until the restaurant changes it to available through the website.
